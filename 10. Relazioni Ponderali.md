I coefficienti stechiometrici forniscono i rapporti in massa tra reagenti e prodotti. Per bilanciare la reazione chimica si deve far si che il numero di atomi sia lo stesso tra i reagenti e i prodotti. Quindi se ho
$$
\ce{CaO + 2HCl -> CaCl_{2} + H_{2}O}
$$Avrò che 1 mole reagisce con 2 moli di HCl perché il numero di moli e i coefficienti stechiometrico sono quasi uguali. Ci danno il rapporto tra il numero di moli dei diversi composti. Il numero di moli non è costante, ma quello che rimane costante è la massa complessiva. Questo è perché nulla deve consumarsi o crearsi nel corso della reazione. 

# ICE
È una tabella in cui si scrive il numero di moli di tutti i composti a diversi step della reazione. 
* I - inizio
* C - change
* E - End
Si parte dai composti chimici 

|     | CaO         | 2HCl        | CaCl$_{2}$  | H$_{2}$O    |
| --- | ----------- | ----------- | ----------- | ----------- |
| I   | 0.1346 mol  | 0.2692 mol  | ---         | ---         |
| C   | -0.1346 mol | -0.2692 mol | +0.1346 mol | +0.1346 mol |
| E   | 0           | 0           | 0.1346 mol  | 0.1346 mol  |

Prima si dovrà calcolare la massa delle molecole
* m$_{\ce{HCl}}$ = n$_{\ce{HCl}}$ $\times$ MM$_{\ce{HCl}}$ = 0.2692 $\times$ 36.461 = 9.815 g
* m$_{\ce{CaCl_{2}}}$ = n$_{\ce{CaCl_{2}}}$ $\times$ MM$_{\ce{CaCl_{2}}}$ = 0.1345 $\times$ 110.984 = 14.94 g
* m$_{\ce{H_{2}O}}$ = n$_{\ce{H_{2}O}}$ $\times$ MM$_{\ce{H_{2}O}}$ = 0.1348 $\times$ 18.015 = 2.425 g

Non 10 cifre significative ma quante servono. 

# Reagente Limitante
Nella maggiorparte delle miscele, vengono messi più reagenti. Quello che viene consumato interamente si chiama reagente limitante. Per determinare il reagente limitante, si prende uno dei due reagenti e suppongo che sia il mio reagente limitante. Il che vuol dire che devo consumare tutte le moli di quel elemento che ho scelto, calcolo la differenza e vedo se mi viene un numero positivo. Il rapporto stechiometrico che c'è non determina niente. 
Esempio:
1. Scelgo HCl come reagente limitante

|     | CaO                         | 2HCl  | ->  | CaCl$_{2}$ | H$_{2}$O |
| --- | --------------------------- | ----- | --- | ---------- | -------- |
| I   | 0.40                        | 0.54  |     | ___        | ___      |
| C   | -0.27 (perché rapporto 1:2) | -0.54 |     | +0.27      | +0.27    |
| E   | 0.13                        | ___   |     | +0.27      | +0.27    |

2. Se invece scegliesse CaO

|     | CaO   | 2HCl  | ->  | CaCl$_{2}$ | H$_{2}$O |
| --- | ----- | ----- | --- | ---------- | -------- |
| I   | 0.40  | 0.54  |     |            |          |
| C   | -0.40 | -0.80 |     |            |          |
| E   | ___   | -0.26 |     |            |          |
